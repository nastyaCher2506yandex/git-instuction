# git-instuction
Описание команд git

## Что такое Git

**Git** — это распределённая система контроля версий (Version Control System, VCS). Её используют, чтобы отслеживать изменения в файлах (чаще всего — в коде), хранить историю правок и удобно работать над проектом в команде.

### Команды Git

---

#### git init

Создание из директории репозитарий. 
---
#### git status

Показывает состояние репозитария.
---
#### git add

Добавление файла в репозитарий для отслеживания. Не имеет данных об изменениях, лишь данные о файлах, которые будут отслеживаться.
---
#### git commit

Сохраняет изменения и описания о них.
---
#### git log

Просмотр истории коммитов.
---
#### git remote

Создание связи между локальным и удаленым репозитариями.
---
#### git remote -v

Проверка соединения репозитариев.
---
#### git push

Добавление изменений в удаленный репозитарий. 
---
#### git clone

Клонирование удаленного репозитария на локальный компьютер.
---
